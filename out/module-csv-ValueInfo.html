Class: ValueInfo

csv~ ValueInfo

new ValueInfo(match, delimiter)

Provides information about a match.
Parameters:
Name Type Description
match string[] A regular expression match. The Regexp in question is one generated by the createRegexp function.
delimiter string The delimiter string used with the createRegexp function.
Properties:
Name Type Description
value string The value part of the match. If the value was originally surrounded by quotes, the quotes will not be present here.
isQuoted boolean Indicates that value was originally surrounded by quotes.
isNewLine boolean Indicates if this is the start of a new row of data.
